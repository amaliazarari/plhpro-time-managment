#Project 53 - Εφαρμογή Διαχείρισης Χρόνου


#Περιγραφή
Εφαρμογή διαχείρισης χρόνου, υποχρεώσεων και ελεύθερου χρόνου, αναπτυγμένη στα πλαίσια του μαθήματος ΠΛΗΠΡΟ του Ελληνικού Ανοιχτού Πανεπιστημίου. Η εφαρμογή επιτρέπει στον χρήστη να καταγράφει δραστηριότητες υποχρεώσεων και ελεύθερου χρόνου, να τις διαχειρίζεται και να βλέπει στατιστικά και βέλτιστη κατανομή χρόνου.

#Εγκατάσταση
1.Κλωνοποίηση του repository
  git clone https://github.com/AmaliaZarari/PLHPRO-Time-Managment.git
  cd PLHPRO-Time-Managment

2.Εγκατάσταση των βιβλιοθηκών
  pip install matplotlib

3.Εκτέλεση
  cd C:\....\src
  python application.py


#Δημιουργία εκτελέσιμου αρχείου .exe
1.Ανοίγουμε το Παράθυρο Κονσόλας

2.Εγκατάσταση PyInstaller
  pip install pyinstaller

3.Δημιουργία .exe
  *Στο φάκελο \src μέσω του terminal τρέχουμε την εντολή:
   pyinstaller --onefile --windowed application.py

   *--onefile: Δημιουργεί ένα μοναδικό εκτελέσιμο αρχείο
   *--windowed: Κρύβει το παράθυρο κονσόλας

4.Εύρεση του application.exe αρχείου
  *Το αρχείου βρίσκεται στο directory  C:\....\src\dist\application.exe

5.Σημείωση
  Μετά τη δημιουργία του .exe αρχείου οι φάκελοι 'dist\' και 'build\' καθώς και το αρχείο 
  με το όνομα 'application.spec' μπορούν να διαγραφούν καθώς δεν χρειάζονται
